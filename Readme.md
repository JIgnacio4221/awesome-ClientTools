## About This Guide

This is **not** a list of terminal emulators. It is a catalog of **CLI utilities** — tools you run *inside* a terminal that let you do remarkable things: generate QR codes, benchmark HTTP APIs, render images inline, explore JSON interactively, monitor GPU usage, stream YouTube music, download from 1000+ sites, manage Kubernetes clusters, and much more.

All tools listed here are free, open source, or have a meaningful free tier. Proprietary tools with no open alternative in their niche are marked accordingly.

## How to Read the Tables

| Field | Description |
|---|---|
| **Name** | Tool name, linked to its homepage or documentation |
| **Summary** | One-line description of what it does |
| **Description** | Key features and typical use cases |
| **OS** | Operating systems it runs on natively |
| **Repository** | Link to the source code |
| **Latest Version** | Stable release — dynamic badge where available |
| **License** | Software license |
| **Open Source** | Whether source code is publicly available |

**OS Abbreviations:** `Win` = Windows · `Lin` = Linux · `mac` = macOS · `All` = all three  
**Legend:** ✅ Yes · ❌ No · ⚠️ Partial or source-available but non-OSI-approved

> Version badges are fetched live from GitHub Releases. Tools hosted elsewhere show a static version or `—`.

## Terminal Multiplexers

Not terminal emulators — these run **inside** any terminal to add persistent sessions, split panes, and window management. Indispensable for SSH workflows.

| Name | Summary | Description | OS | Repository | Latest Version | License | Open Source |
|------|---------|-------------|:--:|------------|:--------------:|---------|:-----------:|
| [tmux](https://github.com/tmux/tmux) | The industry-standard terminal multiplexer | Sessions persist after SSH disconnection, windows split into panes, everything is scriptable. Huge ecosystem of plugins (via TPM) and community dotfile configurations. | Lin / mac / WSL | [tmux/tmux](https://github.com/tmux/tmux) | ![GitHub Release](https://img.shields.io/github/v/release/tmux/tmux?label=) | ISC | ✅ |
| [Zellij](https://zellij.dev/) | Modern, user-friendly multiplexer written in Rust | More approachable than tmux: an always-visible keybinding bar, layout files in KDL format, and a WebAssembly plugin system for extending behavior. | Lin / mac / WSL | [zellij-org/zellij](https://github.com/zellij-org/zellij) | ![GitHub Release](https://img.shields.io/github/v/release/zellij-org/zellij?label=) | MIT | ✅ |
| [GNU Screen](https://www.gnu.org/software/screen/) | The original session multiplexer (since 1987) | Predates tmux and is available on virtually every Unix-like system. Useful on minimal or legacy servers where tmux is not installed. Stable and rarely updated. | Lin / mac / WSL | [savannah.gnu.org/projects/screen](https://savannah.gnu.org/projects/screen/) | — | GPL-3.0 | ✅ |
| [byobu](https://www.byobu.org/) | Tmux/Screen wrapper with status bar | Adds a persistent status bar and keybinding improvements to tmux or GNU Screen. Popular on Ubuntu servers for prettier multiplexing out of the box. | Lin | [byobu/byobu](https://github.com/byobu/byobu) | — | GPL-3.0 | ✅ |
| [tmuxinator](https://github.com/tmuxinator/tmuxinator) | Manage complex tmux sessions from YAML | Define window layouts, panes, and startup commands in YAML. Launch entire development environments with one command. | All | [tmuxinator/tmuxinator](https://github.com/tmuxinator/tmuxinator) | ![GitHub Release](https://img.shields.io/github/v/release/tmuxinator/tmuxinator?label=) | MIT | ✅ |
